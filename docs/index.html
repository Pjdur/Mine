Introduction

Mine is a simple programming language designed to be easy to learn and use. It is suitable for beginners who want to get started with programming, as well as for experienced developers looking for a lightweight language for scripting and automation.

Features

  • Easy to learn syntax
  • Lightweight and fast
  • Cross-platform compatibility
  • Variable assignment
  • Print statements
  • Conditional statements (`if`)
  • Loops (`while`)

Installation

To install Mine, follow these steps:

Download the latest release

  1. Copy the displayed install code for your operating system.
  2. Paste the code into your terminal and run it.
  3. To verify successful installation, run 'mine' in your terminal.

Build from source

  1. Clone the repository: 
    git clone https://github.com/Pjdur/Mine.git
  2. Navigate to the project directory: 
    cd Mine
  3. Build the project: 
    make build
  4. Add the executable to your PATH: 
    export PATH=$PATH:$(pwd)/bin

Loop
